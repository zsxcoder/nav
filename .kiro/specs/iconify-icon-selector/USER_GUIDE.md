# Iconify 图标选择功能使用指南

## 功能概述

Iconify 图标选择功能允许用户在编辑链接时，从 Iconify 的海量图标库中搜索和选择图标。Iconify 提供了超过 150,000 个开源图标，涵盖 Material Design、Font Awesome、Bootstrap Icons 等多个流行图标集。

## 使用方法

### 1. 打开编辑链接对话框

- 点击"添加链接"按钮，或
- 点击现有链接的"编辑"按钮

### 2. 选择图标类型

在"图标"字段中，从下拉菜单选择"Iconify图标"。

### 3. 搜索图标

1. 点击"选择 Iconify 图标"按钮，打开图标选择器
2. 在搜索框中输入关键词（如 "home"、"user"、"settings"）
3. 系统会自动搜索并显示匹配的图标列表

### 4. 选择图标

- 浏览搜索结果
- 点击想要使用的图标
- 图标会自动填充到输入框，并在预览区域显示

### 5. 调整图标效果

- 使用缩放按钮调整图标大小
- 使用颜色选择器更改背景颜色
- 实时预览最终效果

### 6. 保存链接

点击"确定"按钮保存链接。

## 功能特点

### 海量图标库

- 超过 150,000 个图标可供选择
- 涵盖多个流行图标集
- 持续更新的图标库

### 智能搜索

- 实时搜索，快速响应
- 支持模糊匹配
- 搜索结果缓存，提升性能

### 实时预览

- 选择图标后立即预览
- 支持缩放和背景色调整
- 所见即所得

### 状态保持

- 在不同图标类型间切换时，保留之前的选择
- 编辑现有链接时，自动识别 Iconify 图标

## 键盘快捷键

- **Tab**: 在图标选项间导航
- **Enter** 或 **Space**: 选择当前高亮的图标
- **Escape**: 关闭图标选择器

## 常见问题

### Q: 搜索不到想要的图标怎么办？

A: 尝试以下方法：
- 使用更通用的关键词（如用 "arrow" 代替 "arrow-right"）
- 尝试英文关键词
- 使用图标的功能描述（如 "navigation"、"social"）

### Q: 图标加载失败怎么办？

A: 可能的原因和解决方法：
- 检查网络连接
- 确保可以访问 api.iconify.design
- 尝试刷新页面
- 如果问题持续，可以切换到"自定义图标"或"Favicon图标"

### Q: 可以使用自己的图标吗？

A: 可以。选择"自定义图标"类型，然后输入图标的 URL 地址。

### Q: Iconify 图标是免费的吗？

A: 是的。Iconify 提供的所有图标都是开源的，可以免费使用。不同图标集可能有不同的开源许可证，请查看具体图标集的许可信息。

## 技术说明

### 图标 URL 格式

Iconify 图标的 URL 格式为：
```
https://api.iconify.design/{prefix}:{name}.svg
```

例如：
- `https://api.iconify.design/mdi:home.svg`
- `https://api.iconify.design/fa:user.svg`

### 缓存机制

- 搜索结果会缓存 5 分钟
- 减少重复的 API 请求
- 提升搜索响应速度

### 性能优化

- 搜索请求使用节流机制（300ms）
- 图标预览使用懒加载
- 自动清理过期缓存

## 故障排查

### 问题：搜索一直显示"搜索中..."

**可能原因：**
- 网络连接问题
- API 服务暂时不可用

**解决方法：**
1. 检查网络连接
2. 打开浏览器控制台查看错误信息
3. 尝试刷新页面
4. 如果问题持续，联系管理员

### 问题：图标显示为问号或空白

**可能原因：**
- 图标 URL 无效
- 图标加载失败

**解决方法：**
1. 重新选择图标
2. 检查网络连接
3. 尝试使用其他图标

### 问题：搜索结果为空

**可能原因：**
- 关键词不匹配
- 搜索词过于具体

**解决方法：**
1. 使用更通用的关键词
2. 尝试不同的搜索词
3. 检查拼写是否正确
